# AI Productivity Dashboard

A **personal AI productivity system** that monitors your daily notes or work logs, analyzes sentiment and emotions, calculates a productivity score, and visualizes insights.

## Features
- Real-time monitoring of a folder for new text logs
- Sentence-level **sentiment and emotion analysis**
- **Productivity score** calculation
- Emotion distribution charts
- Streamlit web dashboard for interactive visualization
- Works completely offline with open-source AI models
